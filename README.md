## Command Line Interface
- All commands can be listed by using the HELP command.
- To use any specific command, write it down in its given syntax in the command menu.
- You can use either song (or album) name or its serial number, as displayed in the list of the same. Eg. DELETE "abc" is the same as 12 1 (assuming abc is the 1st song in the list)

## Build & run
- Build: `make`
- Run: `make run` or `./c_unplugged`
- Clean build artifacts: `make clean`
